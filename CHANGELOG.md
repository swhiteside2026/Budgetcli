## [0.1.0] - 2026-05-28

### Added

- Six CLI commands: `add`, `summary`, `report`, `list`, `clear`, and `export`
- `--version` flag that prints `budget 0.1.0`
- Local JSON storage via `data/ledger.json` (gitignored, created automatically on first run)
- CSV export via the `export` command
